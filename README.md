# Kerala Ayurveda RAG + Agentic AI System

AI-powered Q&A and content generation system for Kerala Ayurveda, combining Retrieval-Augmented Generation (RAG) with a multi-agent workflow.

## Features

✅ **RAG Q&A System** - Answer customer questions with traceable citations  
✅ **Multi-Agent Blog Generator** - 5-agent pipeline for quality content  
✅ **Adaptive Chunking** - Content-type aware document processing  
✅ **Local Embeddings** - Zero API costs for embeddings  
✅ **Fact-Checking Guardrails** - Prevents hallucinations  
✅ **100% Local** - No deployment dependencies  

## Quick Start

### 1. Install Dependencies
```bash
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Run Demo
```bash
python3 demo.py
```

Choose option 1 for RAG Q&A or option 2 for Blog Generation
